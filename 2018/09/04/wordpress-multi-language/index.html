Wordpress Multi Language

2018-09-04

wordpress
  1. 安装plugin -> WPML
  2. admin->sidebar->settings: e.g. enable translating “Portfolio Categories (portfolio-category)”
  3. 去到Portfolio, 选择一个portfolio，点击 + (Add translation to Chinese) 新建对应的中文portfolio， 然后点击overwrite with English content用英文版本的全部内容填充
